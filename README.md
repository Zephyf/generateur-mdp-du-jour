# Générateur de mots de passe

Petite application Python (Tkinter) qui génère des mots de passe.

Deux fonctionnalités :
- **Générer** : choisir une longueur et des options (majuscules, chiffres, symboles) puis générer un mot de passe.
- **Mot de passe du jour** : une suggestion générée automatiquement chaque jour, stable jusqu'au lendemain, avec un historique des jours précédents.

## Lancer le projet

Nécessite juste Python 3 (Tkinter est inclus par défaut, aucune installation supplémentaire).

```
python main.py
```

## Pourquoi ce projet

Premier projet pour découvrir Python et les interfaces graphiques (Tkinter), dans le cadre de ma préparation à l'option NSI.
